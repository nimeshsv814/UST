# Write a list comprehension that returns all even numbers from 0 to 20.

Answer : [x for x in range(21) if x%2==0]

# Write a list comprehension to extract all words that are longer than 4 characters from a sentences.

Answer : [word for word in sentence.split() if len(word)>4]
